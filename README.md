# PocketGuard - Personal Finance Dashboard

PocketGuard is a simple, responsive personal finance dashboard built using **HTML, CSS, and JavaScript**.  
It allows users to view account balances, see recent transactions, and interact with buttons for sending money, adding funds, requesting money, and navigating between tabs.

---

## Features

- **Balance Overview**: Display current account balance and account number.  
- **Action Buttons**: Top buttons for Send, Add Funds, Request, and More actions.  
- **Transaction List**: Shows recent transactions with name, date, and amount.  
- **Navigation Tabs**: Bottom nav for Home, Wallet, and Settings.  
- **Interactive Alerts**: Clicking buttons shows JavaScript alert messages.
